Implementation of Quotation Tool

I worked on a project to create a Quotation Tool designed to streamline the process of generating quotes for clients. It included features like real-time price updates, cost breakdowns, and the option to send quotes directly through email. The tool was integrated with the company's CRM, making it easier for the team to manage and track quotations efficiently.

ERP System Upgrade and IT Operations Alignment

In this project I worked on upgrading the company’s ERP system and unifying IT operations across multiple departments. The upgrade also involved standardizing IT processes, which led to better resource management and improved cross-department collaboration, ultimately enhancing efficiency throughout the organization.
